Key Insights
Understanding Your Results:
  • Apples-to-apples comparison: Each row compares jobs with the same partition, CPU count, and QoS tier. This ensures a fair comparison.
  • Look at P90/P99: These show the variability and worst-case scenarios. High P90/P99 values indicate unpredictable wait times even if P50 is reasonable.
  • QoS matters: Different QoS tiers (e.g., condo_owner, normal) have dramatically different wait times. Check if you're using the best available QoS for your needs.
  • Recent trends: Use the time window buttons above to see if wait times have changed over time. Compare 30-day (current), 90-day, and 180-day windows.
